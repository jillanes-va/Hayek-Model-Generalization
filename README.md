# Un Modelo de Agentes Basado en la Generalización del Mercado Heurístico de Hayek



Este repositorio contiene el código fuente, el entorno de simulación y las herramientas de análisis de un Modelo Basado en Agentes (ABM) diseñado para generalizar y operacionalizar las intuiciones sobre el sistema de precios como mecanismo de transmisión de información y orden espontáneo, propuestas originalmente por Friedrich Hayek.



El objetivo central de este proyecto es estudiar la emergencia de equilibrios macroeconómicos a partir de interacciones locales descentralizadas, donde agentes con racionalidad limitada y heurísticas adaptativas operan bajo diferentes niveles de fricción institucional e información asimétrica.
